# Course Selection System

A web-based course selection management system developed using SSM (Spring, Spring MVC, MyBatis).

## Technology Stack

- Spring
- Spring MVC
- MyBatis
- MySQL
- JSP
- Maven
- Tomcat

## Features

- User Login
- Course Selection
- Course Management
- Teacher Management

## Database

Import `css.sql` before running the project.
